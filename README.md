# GIFTTED Project Dashboard

Responsive project management dashboard for tracking GIFTTED MVP action items.

## Data Source

The dashboard reads project action items from the Google Sheets `Action Items` tab published as CSV.

## Main Views

- Executive KPI Cards
- Task Status Tracking
- Critical Tasks
- Blocked / Overdue Tasks
- Workstream Progress
- Owner Workload
- Full Action Items Table

## Update Process

Project tasks should be updated only in the Google Sheets `Action Items` tab.  
The dashboard is read-only and refreshes from the published CSV source.
